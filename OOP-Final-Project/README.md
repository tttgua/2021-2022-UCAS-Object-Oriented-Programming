## 游戏规则

###### 基础设定

1. 采用一个矩形地图，地图分为均等的正方形小格，格子的类型有空地，商店，事件，监狱，医院和<空>；
2. 玩家轮流进行自己的回合，通过掷骰子来确定移动格数(玩家抵达不同格子时所触发事件见需求分析部分)；
3. 玩家的财产总额等于现金  + 麾下地产总额(购买加升级)；
4. 当玩家的财产总额不足以缴纳租金时，该玩家宣布破产并退出游戏；
5. 当场上只剩下一位玩家时，游戏结束，该玩家获得胜利。
